# Developer Hub Internship - Phase 2 (Data Science & Machine Learning)

Welcome to my project portfolio for Phase 2 of my Data Science internship at Developers Hub. This repository serves as a centralized hub housing three end-to-end machine learning implementations covering predictive analytics, customer profiling, and temporal time series forecasting.

---

## 📁 Repository Structure & Directory Navigation

Each project is contained within its own dedicated directory, complete with its own granular documentation, clean source data files, visualization distributions, and trained execution models. 

* **[Task 1: Term Deposit Subscription Prediction](./Task%201%20-%20Term%20Deposit%20Prediction/)**
  * *Focus:* Supervised Binary Classification & Explainable AI (XAI).
  * *Objective:* Predict banking product uptake using predictive models (Logistic Regression, Random Forest) and interpret hidden algorithmic weights using SHAP/LIME frameworks.
* **[Task 2: Customer Segmentation Using Unsupervised Learning](./Task%202%20-%20Customer%20Segmentation/)**
  * *Focus:* Unsupervised Clustering & Dimensionality Reduction.
  * *Objective:* Group consumer spending/income habits into 5 distinct behavioral business personas using K-Means Clustering and evaluate structural spreads visually using 2D PCA matrices.
* **[Task 3: Energy Consumption Time Series Forecasting](./Task%203%20-%20Energy%20Consumption%20Forecasting/)**
  * *Focus:* Advanced Temporal Feature Engineering & Forecasting.
  * *Objective:* Model household electrical power usage horizons by comparing classical statistical forecasting (ARIMA) alongside machine learning regressions (Prophet, XGBoost).

---

## 🛠️ Tech Stack & Advanced Libraries Utilized
* **Core Language:** Python 3.x
* **Data Engineering & EDA:** Pandas, NumPy
* **Data Visualization:** Matplotlib, Seaborn
* **Machine Learning Pipelines:** Scikit-Learn
* **Time Series Implementations:** Prophet, XGBoost, Statsmodels
* **Explainable AI Matrix:** SHAP, LIME
* **Model Export Configuration:** Joblib / Pickle serialization formats

## 🚀 Key Learning Milestones Achieved
1. **End-to-End Pipeline Execution:** Architecting complete workflows from raw file ingestion, handling structural disparities, feature engineering, up through finalized model tuning.
2. **Algorithmic Explainability:** Going beyond black-box classification metrics by extracting tangible feature weight interpretations to build cross-functional business trust.
3. **Translating Math into Value:** Transforming unlabelled clusters and complex numerical metrics directly into actionable corporate strategies tailored to targeted consumer groups.
