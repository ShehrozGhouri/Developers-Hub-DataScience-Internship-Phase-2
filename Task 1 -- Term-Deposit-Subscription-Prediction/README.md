# Task 1: Term Deposit Subscription Prediction (Bank Marketing)

## 📌 Objective
Predict whether a bank customer will subscribe to a long-term deposit asset as a direct consequence of an outbound marketing campaign.

## 📊 Dataset
* **Source:** Bank Marketing Dataset (UCI Machine Learning Repository)
* **Target Feature:** `y` (Binary Indicator: 'yes' or 'no')

## 🛠️ Implementation Workflow
1. **Exploratory Data Analysis:** Checked missing records, handled imbalanced class balances, and visualized correlation distributions.
2. **Feature Engineering:** Encoded categorical strings into numerical categories via proper encoding maps (One-Hot / Label encoding).
3. **Model Selection:** Trained classification frameworks, evaluating and comparing performance metrics across baseline models like Logistic Regression and Random Forest Classifiers.
4. **Validation Framework:** Calculated a Confusion Matrix, verified F1-Scores, and mapped the ROC-AUC Curve to assess probability classification thresholds.
5. **Explainable AI (XAI):** Integrated SHAP/LIME analytical values to interpret feature weight allocations and visually explain 5 localized model prediction indices.

## 🚀 Key Skills Demonstrated
* Binary Classification Algorithms
* Performance Metric Evaluation (F1-Score, ROC-AUC)
* Explainable AI (XAI using SHAP/LIME frameworks)
